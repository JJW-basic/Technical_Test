[ Background ]
1. 사용자가 URL 주소와 키워드를 입력하면 키워드에 포함된 정보를 수집하고 정리해 주는 기능을 구현하고자 함.
2. window OS에서 Antigravity IDE의 AI agent를 이용한 바이브 코딩으로 작업을 진행.
3. 01-system-rules.md 파일에 프로젝트 진행 방향성을 명시했음.
4. 프로젝트 초기 환경을 구성하고자 함.


[ User Command ]
나는 사용자가 웹페이지에 들어가서 URL 주소와 관심 키워드를 입력하면 정보를 수집, 정리해 주는 기능을 구현하고자 한다.
기능 구현 전에 로컬에서 원활한 작업을 수행하기 위한 프로젝트 초기 환경을 구성하고자 한다.
UI/UX 디자인, 프론트엔드, 백엔드 이 세 요소를 구분해서 작업을 진행할 수 있어야 한다.
프로젝트 초기 환경을 구성해줘.
